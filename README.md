# 🥣 BOWL 
Bowl (Better OpenGL Wrapper Library) is a library, that offers a variety of advantages over other OpenGL libraries such as Owl.

✅ Blazingly fast ⚡<br>
✅ Lightweight 🍃 <br>
✅ Easy to use 👨‍🎓 <br>
✅ Secure 🔒

## Code Examples
### Creating a simple window
```rust
  let window = bowl::WindowBuilder::new()
        .size(800, 600)
        .fullscreen(false)
        .create();

    window.run(|_handle| {});
```
